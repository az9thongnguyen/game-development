# 2D Lighting Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development or executing-plans.

**Goal:** Additive radial lights: a pure `light_core` (falloff + sample), an additive `Renderer2D::add_pixel` primitive, and a `--light` demo scene.

**Architecture:** Pure `light_core` (`src/engine/fx/light.{hpp,cpp}`) mirrors `particles_core`. `Renderer2D` gains `add_pixel`/`add_cov` (additive twins of blend_pixel/blend_cov). `--light` scene loops each light's bbox and deposits via add_pixel; reuses `tween_core` to drift a light.

**Tech Stack:** C++20, `<cmath>`, no SDL in the core, no new deps.
